# Project Context — Base

## [GENÉRICO] Overview

Descreva seu projeto aqui:
- **Name:** Nome da aplicação
- **Purpose:** O que faz?
- **Users:** Quem usa?
- **Domain:** Em qual domínio? (e-commerce, fintech, etc)

## [GENÉRICO] Problem Statement

O que problema está resolvendo?
- Contexto
- Desafio
- Impacto esperado

## [GENÉRICO] User Personas

Defina 2-3 personas principais:

| Persona | Goals | Pain Points |
|---------|-------|-------------|
| PM | Entregar features rápido | Requisitos mudando |
| Dev | Code limpo | Pressão de deadline |
| User | Experiência fluida | Bugs e crashes |

- Phase 2: Enhanced UX
- Phase 3: Analytics + Growth
- Phase 4: Scale + Performance
